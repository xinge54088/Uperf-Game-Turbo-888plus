# Uperf-Game-Turbo-888plus
需要已加载Uperf-Game-Turbo 性能调度模块 
https://github.com/yinwanxi/Uperf-Game-Turbo/releases

优化888plus 日常体验 兼顾性能和功耗

简介:

    将888plus 的大核频率固定到2.49Ghz 中核频率固定到 2.34Ghz 能耗比最佳频率
  
    调整细节性能调度
  
    实测日常使用体验几乎无卡顿 非常丝滑
  
使用方法 

    把uperf.json 直接复制替换 Android > yc > uperf

    cur_powermode.txt 确保为auto
